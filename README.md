## POKEDEX

Encuentra tus pokemones favoritos y descubre todas sus habilidades 
Si alguna vez pensaste en crear tu propio pokemon, éste es el momento! Puedes crear tu propio pokemon!

### Instalación

- En la carpeta 'api' y 'client'
```js
npm install
```
- Luego
```js
npm start
```
